QUICK PASTE METHOD - DIGITALOCEAN CONSOLE
========================================

Use this if you don't want to install FileZilla!

STEP 1: Open DigitalOcean Console
- Go to your Droplet dashboard
- Click "Access" â†’ "Launch Droplet Console"

STEP 2: Prepare Directories
========================================
cd /var/www/html
mkdir -p api uploads
chmod 775 uploads

STEP 3: Create Files One by One
========================================

FILE 1: registration-form.html
----------------------------
nano registration-form.html

# Press Ctrl+X, then Y, then Enter

FILE 2: dashboard.html
----------------------------
nano dashboard.html
# Paste content from Desktop/registration-system/dashboard.html
# Press Ctrl+X, then Y, then Enter

FILE 3: login.html
----------------------------
nano login.html
# Paste content from Desktop/registration-system/login.html
# Press Ctrl+X, then Y, then Enter

FILE 4: api/submit-registration.php
----------------------------

FILE 5: api/get-registrations.php

----------------------------

STEP 4: Set Permissions
========================================
chown -R www-data:www-data /var/www/html
chmod -R 755 /var/www/html
chmod 775 /var/www/html/uploads

STEP 5: Configure Database (IMPORTANT!)
========================================
nano api/test-connection.php
# Find and change:
# 'username' => 'YOUR_DB_USERNAME'
# 'password' => 'YOUR_DB_PASSWORD'

nano api/submit-registration.php
# Same changes

nano api/get-registrations.php
# Same changes
